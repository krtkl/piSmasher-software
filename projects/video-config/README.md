# HDMI Transmit/Receive Video Configuration

```
Usage: video-config [OPTIONS]

Options:
    -m MODE       - Set video mode (default: WXGA)
    -p PATTERN    - Set test pattern code (default: Color bars)

Modes:
    720p
    1080p
    WXGA

Patterns Codes:
       0          - Video input to test pattern generator
       1          - Horizontal ramp
       2          - Vertical ramp
       3          - Temporal ramp
       4          - Solid red
       5          - Solid green
       6          - Solid blue
       7          - Solid black
       8          - Solid white
       9          - Color bars
      10          - Zone plate
      11          - Tartan bars
      12          - Cross hatch
      13          - Color sweep
      14          - Horizontal and vertical ramp
      15          - Checker
      16          - Pseudorandom
      17          - DisplayPort color ramp
      18          - DisplayPort vertical lines
      19          - DisplayPort color square
```
